Full stack developer!

Hi, I'm Muskan — a web designer and Full Stack Developer. I work with modern front-end and back-end technologies to build responsive, scalable and maintainable web applications.

My core skills include HTML, CSS, Bootstrap and JavaScript for the front-end, along with React to create dynamic user interfaces. On the back-end, I use Node.js and Express.js to handle server-side logic and APIs. For databases, I work with both MongoDB (NoSQL) and SQL(MySQL / phpMyAdmin via XAMPP / Postgresql) for relational data management.

I am also experienced with Git and GitHub for version control, and apply solid Database Management practices to ensure data consistency and scalability.

Beyond coding, I explore Prompt Engineering with ChatGPT and Meta AI to integrate AI-powered solutions and enhance workflows in real-world projects.

With my skillset, I aim to bridge clean design and efficient backend systems, delivering seamless user experiences from idea to deployment.

"I enjoy building modern and responsive web applications using the MERN stack and other front-end technologies. Along with development, I also focus on deployment and hosting. Most of my projects are deployed on Netlify, which allows me to deliver fast, secure, and reliable web experiences accessible from anywhere."
